## **Client**: Walz Mail-Order House  
**Duration**: 2016  
**Objective**:  
Migrate a legacy desktop application to a web-based solution, providing a modern, scalable interface for employees to interact with business workflows.

---

### **Solution**:  
- Analyzed the legacy system and refactored the core business logic for a web-based architecture.  
- Designed and implemented the **backend** using **Spring Boot** to handle business processes efficiently.  
- Created a clean, user-friendly **frontend** with **Vue.js**, offering a responsive and intuitive user interface.  
- Deployed the application in a containerized environment using **Docker** and managed with CI/CD pipelines for continuous delivery.

---

### **Outcome**:  
- Successfully replaced the outdated desktop application with a scalable and maintainable web-based solution.  
- Improved employee productivity through a modern, intuitive interface.  
- Reduced system maintenance costs by consolidating workflows into a streamlined application.  

---

### **Technologies Used**:  
- **Backend**: Spring Boot  
- **Frontend**: Vue.js  
- **Containerization**: Docker  
- **CI/CD**: Jenkins

---

### **Key Contributions**:  
- Refactored legacy code and optimized workflows for modern web-based architecture.  
- Designed and implemented the backend and frontend.  
- Ensured smooth migration with minimal downtime.
