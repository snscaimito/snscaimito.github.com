## **Client**: Mercedes-Benz  
**Duration**: 2021-2022  
**Objective**:  
Simplify the management and operations of SonarQube and GitHub Enterprise for software development and DevOps teams across multiple international locations.

---

### **Solution**:  
- Designed and implemented a backend system using **Java (Quarkus)** to automate and streamline administrative tasks.  
- Built a frontend application with **Vue.js** for an intuitive user interface to manage configurations.  
- Deployed the solution on **Azure Kubernetes Service (AKS)** with CI/CD pipelines to ensure continuous integration and smooth updates.  
- Focused on a robust, scalable design to handle multi-location usage and ensure reliability.

---

### **Outcome**:  
- Improved administrative efficiency for managing SonarQube and GitHub Enterprise configurations.  
- Enabled faster deployments and updates through a streamlined CI/CD pipeline.  
- Enhanced developer productivity by reducing manual intervention and simplifying access management.  

---

### **Technologies Used**:  
- **Backend**: Java (Quarkus)  
- **Frontend**: Vue.js  
- **Containerization**: Kubernetes (AKS)  
- **CI/CD**: GitHub Actions  
- **Infrastructure**: Azure Cloud

---

### **Key Contributions**:  
- Designed the system architecture for automation and scalability.  
- Implemented both backend and frontend components.  
- Integrated deployment pipelines for seamless CI/CD.
