## **Client**: Nationwide Insurance  
**Location**: Columbus, Ohio, USA  
**Duration**: 2009 - 2011  

---

### **Objective**:  
Support an **agile transformation** across 23 teams by providing technical and organizational coaching while solving a specific client need for automating Windows desktop applications.

---

### **Solution**:

#### **Agile Transformation Coaching**  
- Served as a **Senior Developer Advocate / Scrum Coach** supporting a large-scale agile transformation involving **23 teams**.  
- Provided **technical and organizational coaching** to:  
   - **6+ teams** directly, impacting about **80 individuals**.  
   - Iteration Managers and Application Development Leaders to improve leadership practices.  
- Focused on improving team collaboration, agile workflows, and technical delivery through:  
   - Agile practices (Scrum, XP).  
   - Continuous integration and delivery improvements.  
   - Enhanced communication between teams and leadership.  

#### **RAutomation Tool Development**  
- Addressed a specific client need to **automate Windows desktop applications** as part of adopting **Acceptance Test-Driven Development (ATDD)**.  
- Conducted experiments and identified a solution by collaborating with another developer on an open-source project on **GitHub**.  
- Contributed to the creation of **RAutomation**, a small and easy-to-use library for automating Windows controls and applications, with the gracious support of the client.  
- Ensured the tool met the requirements for automating ATDD workflows efficiently and reliably.

---
