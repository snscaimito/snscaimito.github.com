## **Client**: Caimito Agile Life
**Duration**: 2024
**Objective**:  
Develop an IoT sensor dashboard to visualize real-time data and enable monitoring for operational insights.

---

### **Solution**:  
- Designed a cloud-based dashboard application to display real-time data from IoT sensors.  
- Implemented a **backend service** in **Spring Boot** to process and store sensor data.  
- Developed a **frontend** using **React.js** for interactive data visualization and real-time updates.  
- Deployed the solution on **AWS** for scalability and reliability.

---

### **Outcome**:  
- Provided a reliable dashboard to monitor IoT sensor data in real time.  
- Improved operational insights through interactive visualizations.  
- Enabled faster decision-making by offering real-time alerts and trend analysis.

---

### **Technologies Used**:  
- **Backend**: Spring Boot  
- **Frontend**: React.js  
- **Cloud**: AWS  
- **Data Storage**: PostgreSQL  

---

### **Key Contributions**:  
- Architected the system for real-time IoT data ingestion and visualization.  
- Implemented backend data processing and frontend visualizations.  
- Ensured a scalable deployment on AWS infrastructure.
