## **Client**: Serials Solutions  
**Location**: Seattle, Washington, USA (Geographically Distributed Team)  
**Duration**: 2008 - 2009  

---

### **Objective**:  
Collaborate within a **self-organizing, geographically distributed team** to co-create the **Summon product**, a discovery service for academic and research libraries.

---

### **Solution**:  
- Worked as part of a **6-person international team** to design and develop core features of the **Summon product**.  
- Focused on building a scalable and performant search platform leveraging **Apache Solr** and **Apache Lucene** for indexing and querying large volumes of library metadata.  
- Developed robust data pipelines using:  
   - **Apache Camel** for integration and routing of XML data.  
   - **ActiveMQ** for reliable messaging and event-driven workflows.  
- Implemented XML-based data transformations to process and normalize content from various sources into a searchable index.  
- Ensured high performance and search accuracy through iterative development, testing, and optimization.

---

### **Outcome**:  
- Successfully co-created the **Summon product**, providing libraries with a powerful, scalable search and discovery tool for their collections.  
- Delivered a performant search platform capable of indexing and querying large-scale metadata efficiently.  
- Enabled seamless integration of diverse data sources using messaging and routing technologies.  
- Contributed to a **self-organizing, geographically distributed team**, demonstrating effective collaboration and autonomy.

---

### **Technologies Used**:  
- **Languages**: Java  
- **Search Technologies**: Apache Solr, Apache Lucene  
- **Data Integration**: XML, Apache Camel  
- **Messaging**: ActiveMQ  

---

### **Key Contributions**:  
- Co-developed core components of the **Summon product** as part of an international 6-person team.  
- Built scalable search functionality using **Apache Solr** and **Apache Lucene**.  
- Designed and implemented data processing pipelines using **Apache Camel** and **ActiveMQ**.  
- Enabled seamless integration of XML-based data sources into a unified search index.
